# Creating a community

1. **Navigate to “New Community”**:
   Go to the Community section using the cm command, and click on “+ New Community”.

2. **Add Details:**
   Provide a name and description so others understand what your community is about.

3. **Invite Members:**
   Add the users you wish to include by entering their usernames or email addresses.

:::info
You can add in users with their username, email, or wallet address linked to their PANDA Terminal account. Only users with an account on PANDA Terminal can be added.
:::

4. **Complete Creation:**
   Click “Create” to successfully set up your new community.

Your community is successfully set up!
