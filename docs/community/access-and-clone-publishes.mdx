# Access and Clone Publishes

## Access Community Publications

- Navigate to the `cm` route.

- Go to Joined Communities and select the desired community page.

## View Published Ideas

- In the Published section, explore all the shared ideas within the community.

## Clone Templates

- To use a published template, view the publish, and click “Clone”.

:::info
Cloning creates a personal working copy, independent of the original idea.
Any changes you make to the cloned version will not affect the original. You are
free to analyze and build upon the ideas shared.
:::
