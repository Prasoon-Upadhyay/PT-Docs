# Sharing in Workbench

Follow these simple steps to share a workbench.

1.  Select the workbench view that you need to share, and activate it on your chart.
2.  Navigate to the drop down next to the view title, and enable Sharing.
3.  Add the PANDA Terminal users that you want to share with.
