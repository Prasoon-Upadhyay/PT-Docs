# Sharing in PANDA Intel

Follow these simple steps to share a PANDA Intel.

- Select the PANDA Intel that you need to share, and activate it on your chart.
- Edit the PANDA Intel settings, and enable Sharing.
- Add the PANDA Terminal users that you want to share with.
