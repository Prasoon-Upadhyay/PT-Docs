# Navigating the Chart's Interface

The charting section is the customizable chart area integrated with intelligent drawing tools that provide professional charting capabilities. It is the main component of the workbench and includes the following elements, starting from the top-left:

- **Symbol Lookup** - The market and token symbol selector
- Charting Tools, Drawing & Duration selectors
- Cross hairs & Info Display
- Chart Type Menu
- **Views Menu** - Collection of your saved views
- **Studies Menu** - Variety of Metrics for analysis
- **Events Menu**- PANDA Intel or Signal indicators for the chart
- Filters for chart configuration
