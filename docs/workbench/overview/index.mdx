# Overview

The Workbench is the default home page upon logging in.

## What is the Workbench?

The Workbench offers a live data feed for tracking various tokens, combining historical and real-time data for in-depth analysis. It integrates custom-built metrics designed to uncover alpha, helping users analyze markets more effectively.
