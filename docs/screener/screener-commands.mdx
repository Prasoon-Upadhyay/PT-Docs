# Screener Commands

The base command for all workbench operations is prefixed by `sc`.

The following tables list all the commands used inside the screener, categorized by their usage.

| Command               | Description                       |
| --------------------- | --------------------------------- |
| `sc`                  | Primary Command for screener      |
| `sc name`             | Navigate to a saved screener view |
| `sc add default name` | Add pre-saved screener view       |
