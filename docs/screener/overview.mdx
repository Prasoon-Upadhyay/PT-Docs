# Overview

The Screener in PANDA Terminal simplifies market exploration.

Use the `sc` command to switch to the screener module and access default views like market data, on-chain data, and DEX screeners.

It displays tokens based on on-chain and off-chain metrics in a clear table, helping you:

- Identify trending tokens
- Monitor on-chain and off-chain activity
- Analyze token performance across the market.
