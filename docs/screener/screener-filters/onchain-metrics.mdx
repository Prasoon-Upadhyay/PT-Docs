# Onchain Metrics

On-chain metrics provide blockchain-specific data that reflect network activity and token flow. These metrics include:

## Gini

Measures the inequality in the distribution of tokens across addresses.

## Velocity

The rate at which tokens move between addresses.

## Active Addresses

The total number of addresses involved in token transactions.

## Active Senders

The count of unique addresses sending tokens.

## Active Receivers

The count of unique addresses receiving tokens.

## On-chain Volume

The total amount of tokens transferred on the blockchain over a specified period.
