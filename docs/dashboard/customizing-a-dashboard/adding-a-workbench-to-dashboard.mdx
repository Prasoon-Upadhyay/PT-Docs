# Adding a Workbench to Dashboard

The Workbench lists all your saved workbenches, serving as a central hub for accessing and organizing your customized tools and data configurations. You can easily switch between them to suit your current task.

Follow these simple steps to add your first workbench into the dashboard:

<Tabs>
  <TabItem value="cmd" label="Add wb using command" default>
    <ul>
      <li>
        <b>Open the Terminal</b>
        <p>
          Access the terminal where all dashboard management commands are
          entered.
        </p>
      </li>
      <li>
        <b>Enter the Command</b>:
        <p>Type the following command to create a new dashboard:</p>
        <p>
          <code>db add wb [name-of-the-workbench]</code>
        </p>
      </li>
      <li>
        <b>Enter your name</b>:
        <p>
          Type the name that you want to import the saved workbench view as, and
          Confirm.
        </p>
      </li>
    </ul>
    <video width="800" height="500" autoPlay loop>
      <source
        src="https://assets.pandaterminal.com/documentation_clips/adding%20wb%20using%20command%20in%20db.mov"
        type="video/mp4"
      ></source>
    </video>
  </TabItem>
  <TabItem value="gui" label="Add wb using the GUI">
    <ol>
      <li>
        Click on <strong>“Add Widget”</strong>.
      </li>
      <li>
        Navigate to <strong>“Workbench”</strong> and select the name of the
        workbench from your current workbench. Also, add the Widget name.
        <ul>
          <li>
            Note: The module is set to <strong>“Workbench”</strong> by default.
          </li>
        </ul>
      </li>
      <li>
        Click <strong>Add</strong>.
      </li>
      <li>Now that the Workbench is visible on the Dashboard.</li>
    </ol>
    <video width="800" height="500" autoPlay loop>
      <source
        src="https://assets.pandaterminal.com/documentation_clips/adding%20wb%20in%20db.mov"
        type="video/mp4"
      ></source>
    </video>
  </TabItem>
</Tabs>
Your workbench is successfully added into your dashboard.
