# Adding a Screener to Dashboard

Follow these simple steps to add your first screener into the dashboard:

<Tabs>
  <TabItem value="cmd" label="Add sc using command" default>
    <ul>
      <li>
        <b>Open the Terminal</b>
        <p>
          Access the terminal where all screener management commands are
          entered.
        </p>
      </li>
      <li>
        <b>Enter the Command</b>:
        <p>Type the following command to add a screener to your dashboard:</p>
        <p>
          <code>db add sc [name-of-the-screener]</code>
        </p>
      </li>
      <li>
        <b>Confirmation </b>:
        <p>
          Confirm in the GUI regarding the screener you want to add to the
          dashboard.
        </p>
      </li>
      <li>
        <b>Click Add </b>:
        <p>The screener will now be visible in your dashboard.</p>
      </li>
    </ul>
    <video width="800" height="500" autoPlay loop>
      <source
        src="https://assets.pandaterminal.com/documentation_clips/adding%20sc%20using%20command.mov"
        type="video/mp4"
      ></source>
    </video>
  </TabItem>
  <TabItem value="gui" label="Add sc using the GUI">
    <ul>
      <li>
        <b>Click on “Add Widget”</b>
        <p>Open the dashboard and click on the “Add Widget” button.</p>
      </li>
      <li>
        <b>Navigate to “Screeners”</b>
        <p>
          Select the “Screeners” option from the available widgets, then choose
          the screener name from your saved screeners.
        </p>
      </li>
      <li>
        <b>Widget Setup </b>
        <p>
          The module will automatically be set to “Screener.” Add a name for the
          widget if required.
        </p>
      </li>
      <li>
        <b>Click Add </b>
        <p>The screener will now be visible in your dashboard.</p>
      </li>
    </ul>
    <video width="800" height="500" autoPlay loop>
      <source
        src="https://assets.pandaterminal.com/documentation_clips/adding%20sc%20in%20db%20using%20gui.mov"
        type="video/mp4"
      ></source>
    </video>
  </TabItem>
</Tabs>
