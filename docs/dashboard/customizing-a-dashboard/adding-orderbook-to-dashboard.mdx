# Adding Orderbook to Dashboard

The Orderbook module allows you to monitor transactions related to a specific entity or wallet address. You can set time intervals (e.g., 1W for one week, 1M for one month) to view token movements within a selected period.

The data is updated in real time, giving you accurate insights into transactions happening within the specified timeframe.

The orderbook’s live data view helps you keep track of in-and-out token movements for a wallet on the same day.
