# Adding Trades to Dashboard

The Trade Lists module provides a comprehensive view of trading activity and the transaction history for a specific entity, wallet, or exchange account.

This feature helps you track all historical trades, making it easier to analyze performance over time.

The trade lists are highly customizable, enabling you to filter based on time periods or specific transaction types.
