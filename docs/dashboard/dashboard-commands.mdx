# Dashboard Commands

The base command for all dashboard operations is prefixed by `db`.

The following tables list all the commands used inside the dashboard, categorized by their usage.

| Command               | Description                        |
| --------------------- | ---------------------------------- |
| `db`                  | Primary Command for dashboard      |
| `db name`             | Navigate to a saved workbench view |
| `db add default name` | Add pre-saved dashboard view       |
| `db add sc name`      | Add a saved screener to a view     |
| `db add wb name`      | Add a saved workbench to a view    |
| `db add orderbook`    | Save orderbook to a view           |
| `db add trades`       | Resave the active workbench        |
| `wb clear`            | Clear the active workbench         |
